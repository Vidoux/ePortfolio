Site vitrine | Harry Poter

2019 - 1 Semestre

 Previous Next
Objectif
Développer un site Web en HTML5 CSS3 et Javascript
Créer du contenu multimédia à travers un projet
Outils
HTML5, CSS3, Gimp, sublimText, Axure
Description rapide
L’objectif du projet est de réaliser un site internet aillant pour thème général un film. La maitrise d’ouvrage a choisi le film The Greatest Showman. Ce projet est réalisé à l’UTT dans le cadre de l’UE MM01. Dans ce projet il y aura 5 intervenants : le chef de projet, le client, la graphiste, le développeur et l’ergonome.
Compétences
  • Gestion de projet
  • concevoir un graphisme adpté au contenu et agréable
  • Déveloper ses capacité en codage HTML CSS
